## About the Program

The [MS in Data Science at CUNY SPS](https://sps.cuny.edu/academics/graduate/master-science-data-science-ms) is a 30-credit, fully online program ranked by *Fortune* as one of the best online master's programs in the nation. The curriculum is built around hands-on, project-based learning using Python and R, with an emphasis on real-world applications in machine learning, artificial intelligence, business intelligence, and data governance.

### Degree Requirements (30 Credits)

| Category | Credits |
|---|---|
| Required Core Courses (4 courses) | 12 |
| Elective I — Analytics Courses (2 courses) | 6 |
| Elective II — Specialty Courses (2 courses) | 6 |
| Capstone | 6 |
| **Total** | **30** |

### Core Courses
- **DATA 602** — Introduction to Data Analysis & Visualization
- **DATA 606** — Statistics & Probability for Data Analytics
- **DATA 607** — Data Acquisition & Management
- **DATA 608** — Knowledge & Visual Analytics
